Building a Diffusion Model from Scratch

This project focuses on building a diffusion model from scratch to understand how modern generative AI systems create data. A diffusion model works by gradually adding noise to data and then learning how to reverse that process to reconstruct meaningful outputs.

The goal of this project was to explore the core idea behind popular AI models used in image generation and to implement the complete workflow, including noise addition, denoising, and model training.
